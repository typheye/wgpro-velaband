# wgpro-velawatch

## 项目介绍
该版本腕管Pro适用于搭载Vela系统的小米智能手环设备运行。

## 适配情况

| 功能列表 | Xiaomi Band 9系列 | Xiaomi Band 10系列 |
| :----  | ----:  | ----: |
| 基础功能 | 支持 | 支持 |
| 系统信息 | 支持 | 支持 |
| 存储空间 | 支持 | 支持 |
| 显示测试 | 支持 | 支持 |
| 腕上微聊 |  |  |
| 应用更新 |  |  |
| Typheye账户 |  |  |

注：空白项即还未经过测试。
## 项目编译

### 1. 开发

```
npm install
npm run start
```

### 2. 构建

```
npm run build
npm run release
```

### 3. 调试

```
npm run watch
```
